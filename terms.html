ChatGPT & Gemini PDF Exporter is a browser extension for exporting supported ChatGPT and Gemini pages to PDF print preview or Markdown files. It is not affiliated with OpenAI, ChatGPT, Google, or Gemini.

PDF export relies on Chrome print preview and your operating system save dialog. The extension cannot guarantee that Chrome or the operating system will expose the same save flow on every platform.

Markdown export saves files through the browser downloads API.
