Privacy Policy

ChatGPT & Gemini PDF Exporter exports supported AI conversations to PDF print preview or Markdown files inside your browser. It is not affiliated with OpenAI, ChatGPT, Google, or Gemini.

Data We Process

The extension may process the following data:

  • Conversation text and HTML from supported ChatGPT or Gemini pages.
  • Selected text or selected conversation turns you choose to export.
  • The current page title and URL, used as source information in exported documents.
  • Temporary export jobs used to open the PDF print preview.
  • Local UI state such as selected export mode.
  • Google account id and email when you sign in for Supporter License activation.
  • License tier, license status, payment provider customer/order ids, refund status, and cancellation status.

Local Processing

Conversation content is converted in the browser. The extension does not send conversation content to our server or any third-party analytics service.

PDF export uses Chrome print preview. Markdown export uses the Chrome downloads API to save files on your device.

Google Sign-In Purpose

Google sign-in is used only to identify your account for Supporter License activation, checkout, license refresh, cross-device restoration, refunds, and cancellations.

Google sign-in is not required for basic PDF export, basic Markdown export, Total auto-collection, local downloads, or reading supported page content in the browser.

Storage Location

Temporary export jobs are stored with Chrome extension storage while the export flow is running. Markdown files and PDF files are saved only where you choose in your browser or operating system save dialog.

External Transfers

The extension does not transfer conversation content to external servers, does not run analytics, and does not sell user data.

For Supporter License purchases, account and payment status data may be shared with Lemon Squeezy and our license server only to process payment, activate the license, handle refunds, and provide support.

User Control

You choose when to export, which mode to use, and where files are saved. You can remove the extension from Chrome at any time.
